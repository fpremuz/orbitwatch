# Communication Loss

## Symptoms

- Signal strength below threshold
- Missing telemetry packets
- Extended communication gaps

## Possible Causes

- Antenna misalignment
- RF subsystem failure
- Atmospheric interference
- Ground station outage

## Risk Level

Medium

## Recommended Actions

1. Verify antenna orientation.
2. Check RF subsystem status.
3. Retry communication windows.
4. Escalate if outage exceeds mission limits.
