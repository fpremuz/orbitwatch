# Orbit Deviation

## Symptoms

- Orbit deviation exceeds expected limits
- Altitude drift
- Unexpected orbital behavior

## Possible Causes

- Thruster malfunction
- External collision event
- Gravitational perturbation
- Navigation software issue

## Risk Level

Medium

## Recommended Actions

1. Review navigation telemetry.
2. Verify thruster health.
3. Calculate correction maneuver.
4. Monitor future orbital updates.
